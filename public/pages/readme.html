README

Sobre o Projeto

Forca na Fenda é um jogo multiplayer da forca com poderes, sistema de apostas, loja, progressão de campanha, ranking e sincronização em tempo real via WebSockets. O jogo foi criado como projeto acadêmico e apresenta diversas telas, modos e sistemas integrados.

Fluxo Completo para Jogar

O fluxo principal de navegação é:

  • Menu inicial
  • Seleção de modo
  • Criação de sala (categorias)
  • Host / Guest
  • Preparação (poderes e apostas)
  • Gameplay sincronizado
  • Vitória ou derrota

Funcionalidades

  • É divertido!
  • Multiplayer via WebSocket
  • Monetização via sistema de poderes, com compra e uso
  • Apostas entre jogadores
  • Loja com inventário unitário
  • Modo campanha: para saber mais sobre os integrantes do projeto!
  • Ranking global
  • Autenticação (login e cadastro)
  • Seleção de categorias

Modo Campanha

O modo campanha apresenta fases especiais baseadas nos integrantes do projeto. Cada fase possui:

  • História própria
  • Palavras temáticas
  • Progresso salvo no banco
  • Recompensas ao completar

Tecnologias

  • Node.js
  • Express
  • WebSocket Nativo
  • Postgres + Sequelize
  • Supabase + Railway
  • HTML / CSS / JS
  • Animações personalizadas

Como Rodar

Instale dependências:

npm install

Execute o servidor:

npm start
